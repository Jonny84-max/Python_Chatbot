## Python_Chatbot
A simple chatbot that greets users and reminds them of today's date and time in WAT

## Features
Accepts user name as input  
Generates personalized greeting  
Displays current date and time (WAT)  
Simple and user-friendly interface  
Deployed for online access  

## Live Demo
